#  Notes App (Full Stack)

A **Google Keep-like Notes Application** built using **Spring Boot + MySQL (Backend)** and **React + Tailwind CSS (Frontend)**.

This project demonstrates **REST API development, JPA relationships, pagination, and frontend-backend integration.**

---

##  Features

*  Create notes
*  Update notes
*  Delete notes
*  View notes with pagination
*  One-to-many relationship (**Notes ‚Üí Tags**)
* REST API integration
* Responsive UI with Tailwind CSS

---
